# Finance-Tracker-App

A full-stack web application built with Flask and SQLite that allows users to add, store, and view personal expenses through a simple web interface.

This project demonstrates backend development, database integration, and frontend rendering using Python, HTML, and CSS.

---

## Features

- Add new expenses with description and amount  
- Persist data using an SQLite database  
- View all recorded expenses in a dynamic table  
- Simple and clean user interface  
- Server-side rendering with Flask and Jinja2  

---

## Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, CSS  
- **Tools:** VS Code
